# Rule: Project Hard Rules — baza_bot

Выстрадано 13 кругами `/review-plan`. Это не рекомендации — это места, где код молча ломается или бот врёт агенту. Нарушение любого = red-review.

---

## Данные и источники

1. **Проект — из реестра, не из пути файла.** `server/registry/projects.ts` — единственный источник атрибуции. Угадывание по строке пути теряет ЖК (прототип отправил ДНК в Бестселлер). Цена: неверный процент вознаграждения.

2. **Источник истины по условиям — партнёрский портал `realtors.baza.bz`**, не таблица тест-кейсов. По Алисе они расходятся (портал 2,6%+1%, таблица 2.7%) — прав портал (решение владельца). Таблица тест-кейсов = только триггеры и фолбэк.

3. **Комиссия — структура `{base, bonus, bonusRule}`, не число.** Агенту дают 1% на выбор: себе в вознаграждение или скидкой клиенту, только один вариант. Ответ обязан содержать механику выбора.

4. **Дисклеймер обязателен.** При любом ответе про комиссию/ипотеку/рассрочку/акции — «*Не оферта. Акции не суммируются. Точные условия уточняйте в отделе продаж.*» Бот не публикует оферту от имени застройщика.

5. **Таблица тест-кейсов — техзадание, а не контент.** Колонка «Пример ответа» смешивает готовые ответы, заметки разработчику («я думаю, надо переводить») и инструкции формата («Нужно продуктово отвечать так»). Canned-тексты **пишутся заново** по ячейкам как по спеке. Сидер падает, если у группы нет выверенного текста.

## Парсинг (ingest)

6. **Yandex Disk отдаёт имена в NFD.** Буква «й» ломает матч по имени (`знаний`, `развлечений`). Нормализовать `unicodedata.normalize('NFC', …)` с обеих сторон, иначе источники выпадут молча.

7. **Tilda: удалить `style/script` до `innerText`.** Иначе innerText тащит инлайновый CSS (один блок = 72k символов CSS вместо текста). Брать только верхнеуровневые `.t-rec` + дедуп.

8. **best.baza.bz и портал — только headed chromium.** best отдаёт headless-браузеру заглушку BotFAQtor (314 симв). Портал троттлит **403** при быстрых запросах — паузы ~1s между страницами. Ingest в проде идёт через xvfb.

9. **Ingest тянет из URL, не из локальных копий.** ТЗ требует повторяемости — «перезапустил, база обновилась». Локальные файлы = запрещённая копипаста.

10. **Сканы отсеиваются громко.** PDF без текстового слоя (< N симв/страницу) → пометка `needs_ocr` в `IngestRun`, не пустой чанк (NO FALLBACKS).

## БД

11. **`prisma db pull` запрещён.** Схема содержит `Unsupported("vector(1024)")` — db pull сгенерирует DROP колонки embedding. Миграцию с `CREATE EXTENSION vector` и `vector(1024)` править вручную.

12. **Prisma 7 требует driver adapter** (`@prisma/adapter-pg`). ESM-only, `import "dotenv/config"` явно.

13. **Локальный порт Postgres — 5433** (5432 занят `postgresql@18` на машине разработки).

## Внешние API

14. **Голосовые: байты через Soniox `/v1/files`, НЕ `audio_url`.** Telegram-ссылка содержит токен бота — отдать её Soniox = отдать бота. Опция `api_key` (не `apiKey`). Проверять `status` (ошибки асинхронные, `/transcript` на упавшей задаче → 409).

15. **Числа и единицы цитируются дословно, не пересказываются.** Замер поймал «219 м-м» (машиноместа) → «219 м²» и выдуманные потолки ДМД2 из чужого ЖК.

## Ответы бота

16. **Агентские проекты (Сезоны, Кварталы, River Park, МК) ≠ «не знаю».** У них нет презентаций, потому что BAZA их продаёт как агентство. Ответ: «BAZA реализует как агентство, КВ 1%» + менеджер.

17. **Приоритет: интент > проект.** «Сколько первоначалка на ДМД2» → canned про рассрочку, а не пересказ корпуса ДМД2.

18. **Менеджер и canned — по региону проекта.** ЕКБ → Яна Крылосова (не «Крылосовла» из таблицы) + `/ekb`; МСК → Алёна Дружинина + `/msk`. Регион неизвестен → оба контакта.
